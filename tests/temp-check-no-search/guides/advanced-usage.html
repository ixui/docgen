高度な使用方法

DocGenの高度な機能について説明します。

テーマのカスタマイズ

CSSのカスタマイズ

独自のCSSを作成できます：

.custom-header {
  background-color: #2196F3;
  color: white;
}

テンプレートのカスタマイズ

Handlebarsテンプレートを使用してカスタマイズ：

<div class="custom-layout">
  <header>{{metadata.title}}</header>
  <main>{{{content}}}</main>
</div>

バッチ処理

複数ファイルの処理

docgen build docs/ --output dist/

ウォッチモード

docgen build docs/ --watch
