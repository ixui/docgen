DocGen テストドキュメント

これはDocGenのテスト用ドキュメントセットです。

概要

このドキュメントセットには以下が含まれています：

  • ガイド - 使用方法のガイド
  • API - API リファレンス
  • チュートリアル - ステップバイステップのチュートリアル

はじめに

DocGenを使って効率的にドキュメントを生成しましょう。

インストール

npm install -g docgen

基本的な使用方法

docgen build docs/ -o output/
